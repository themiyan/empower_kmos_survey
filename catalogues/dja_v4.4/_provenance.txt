Key spec-z columns:
  ra, dec    : astrometry (deg), Gaia/NIRCam-tied, sub-pixel
  z_best     : best spectroscopic redshift
  grade      : confidence grade 0-3 (3 = secure; downstream uses grade==3 & z_best>0)
  + emission-line fluxes (see .columns.csv)

Notes:
  Previously downloaded at runtime from Zenodo via astropy download_file(cache=True).
  Now stored locally to remove the network dependency; load_dja_nirspec() in the
  spec-z analysis reads this file directly. Match radius 0.5" (SPECZ_MATCH_RADII).
  In the GOODS-S layer priority, JADES-DR4 sits above DJA.
